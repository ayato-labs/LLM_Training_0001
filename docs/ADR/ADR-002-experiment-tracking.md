# ADR-002: 実験管理ツール(TensorBoard)の導入
- 日付: 2026-06-23
- ステータス: 実装済み

## 背景
学習のロス推移などが自動記録されておらず、実験の定量的な比較ができていない。また、ビジネス利用を考慮し、外部SaaSへの依存を避け、オープンソースライセンス（MIT/Apache 2.0）に基づくツールでローカル環境で完結させる必要がある。

## 決定
TensorBoard を導入し、学習実験のロスや学習過程をローカル環境に自動的に記録する。

## 理由
外部へのデータ送信なしで完結し、オープンソースライセンスで利用可能であるため。また、TransformersのTrainerとネイティブに統合されており、追加の開発負荷を抑えつつ、トレーサビリティを向上できる。
