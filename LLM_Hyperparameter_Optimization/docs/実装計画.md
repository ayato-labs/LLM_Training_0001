## 理論的背景（再現に必要な最小限）

### Step Law（2025年3月, StepFun）

3,700 LLM を実際に学習し、約100万 H800 GPU 時間を消費して導出したべき乗則。

$$
\eta^*(N, D) = 1.79 \cdot N^{-0.713} \cdot D^{0.307}
$$
$$
B^*(D) = 0.58 \cdot D^{0.571}
$$

- $N$：パラメータ数、$D$：学習トークン数
- モデルの幅・深さ・FFN比率に依存しない（Topological Invariance）
- Dense / MoE どちらにも適用可能
- コード・データ・チェックポイントが全公開（`github.com/step-law/steplaw`）

**重要な副次知見：**  
min_lr を `max_lr / 10`（旧来）に設定すると最適 LR が左方向（小さい LR）にシフトし、loss が悪化する。  
→ `min_lr = 1e-5`（固定定数）の設定が推奨。

### NorMuon（2025年10月）

Muon（MomentUm Orthogonalized by Newton-Schulz）に neuron-wise adaptive LR を追加したオプティマイザ。

```
G_t → Newton-Schulz 直交化（7ステップ）→ NeuronNorm → 更新
```

IMU-1 の ablation（70M Proxy）では AdamW 比 **-2.88% loss**（アーキテクチャ固定）。  
Cautious Weight Decay（gradient と weight の符号が一致する場合のみ decay を適用）を追加すると **-3.85%**。

2D パラメータ（重み行列）: NorMuon `lr=0.0235`  
1D パラメータ（embedding, bias, LN）: AdamW `lr=0.007`

### WSD スケジュール + 1-√t decay

```
[Warmup: 1-2%] → [Stable: 80%] → [Decay: 20%]
                                    1 - √(t/T_decay)
```

IMU-1 の ablation では WSD(20% decay) = Cosine と同等 loss だが、  
**安定フェーズのチェックポイントから追加データで継続学習できる**という決定的な運用上の優位性がある。

Stable フェーズの LR = Cosine peak LR × 55% が実験的に最適。

### IMU-1 レシピ（2026年1月, 公開再現可能）

430M パラメータ、72B トークンで SmolLM2-360M（4T トークン）の 98% の性能を達成。

**アーキテクチャ変更の効果（70M Proxy ablation）：**

| 変更 | 単独効果 |
|------|---------|
| QK-Norm | -0.63% loss |
| Normalized Value Residual | -0.38% |
| LayerNorm Scaling（深さ補正） | -0.36% |
| Per-Head Gating | -0.05% |
| **全部合計** | **-1.64%（相乗効果）** |

全変更 + NorMuon + CWD で合計 **-5.21% loss**（一致トークン・ステップ数比較）。

## Phase 0-B：Step Law によるHPO事前計算

**期間目安：1 週間**  
**目標：グリッドサーチなしに最適 LR・Batch Size を計算する**

### B-1. Step Law 計算機の実装

```python
# utils/step_law.py
# 論文: arXiv:2503.04715, 数式は Step Law 公式サイトより

import math

def step_law_optimal_lr(n_params: int, n_tokens: int) -> float:
    """
    Step Law による最適 Learning Rate の計算。
    
    式: η*(N, D) = 1.79 * N^(-0.713) * D^(0.307)
    
    注意:
      - N: embedding を除く非語彙パラメータ数
      - D: 学習トークン数
      - min_lr は max_lr/10 ではなく 1e-5 (固定) を使うこと
    
    Args:
        n_params: モデルの（非語彙）パラメータ数
        n_tokens: 学習に使用するトークン数
    
    Returns:
        最適 max_lr の推定値
    """
    return 1.79 * (n_params ** -0.713) * (n_tokens ** 0.307)


def step_law_optimal_batch(n_tokens: int) -> int:
    """
    Step Law による最適 Batch Size（トークン数単位）の計算。
    
    式: B*(D) = 0.58 * D^(0.571)
    
    Args:
        n_tokens: 学習に使用するトークン数
    
    Returns:
        最適 batch size（トークン数単位）
    """
    b_tokens = 0.58 * (n_tokens ** 0.571)
    return int(b_tokens)


def compute_hpo_for_target(
    n_params: int,
    n_tokens: int,
    seq_len: int = 512,
) -> dict:
    """
    ターゲットモデルの HPO を Step Law から一発計算。
    グリッドサーチ不要。
    
    Args:
        n_params: パラメータ数（embedding 除く）
        n_tokens: 学習トークン数
        seq_len:  シーケンス長
    
    Returns:
        HPO 設定の辞書
    """
    max_lr = step_law_optimal_lr(n_params, n_tokens)
    batch_tokens = step_law_optimal_batch(n_tokens)
    batch_seqs = max(1, batch_tokens // seq_len)

    # Muon LR / AdamW 1D LR の推奨比率（IMU-1 実験より）
    muon_lr  = max_lr
    adamw_lr = max_lr * (0.007 / 0.0235)   # ≈ max_lr * 0.298

    return {
        'n_params'    : n_params,
        'n_tokens'    : n_tokens,
        'seq_len'     : seq_len,
        'max_lr_2d'   : round(muon_lr, 6),   # NorMuon / Muon 用
        'max_lr_1d'   : round(adamw_lr, 6),  # AdamW (embedding, bias, LN) 用
        'min_lr'      : 1e-5,                # ★ 固定（Step Law 推奨）
        'batch_size_tokens': batch_tokens,
        'batch_size_seqs'  : batch_seqs,
        'stable_lr_ratio'  : 0.55,           # IMU-1: stable LR = peak × 0.55
        'stable_lr_2d'     : round(muon_lr * 0.55, 6),
    }


# ─── 使用例 ───────────────────────────────────────────────────
if __name__ == '__main__':
    configs = [
        ('70M Proxy',  70_000_000,   1_400_000_000),   # IMU-1 ablation と同スケール
        ('125M',      125_000_000,  10_000_000_000),
        ('350M',      350_000_000,  25_000_000_000),
    ]

    for label, n, d in configs:
        hpo = compute_hpo_for_target(n, d)
        print(f"\n── {label} ({n/1e6:.0f}M params, {d/1e9:.1f}B tokens) ──")
        print(f"  max_lr (NorMuon 2D) : {hpo['max_lr_2d']:.6f}")
        print(f"  max_lr (AdamW 1D)   : {hpo['max_lr_1d']:.6f}")
        print(f"  stable_lr (NorMuon) : {hpo['stable_lr_2d']:.6f}")
        print(f"  min_lr              : {hpo['min_lr']:.0e}  ← 固定")
        print(f"  batch (tokens)      : {hpo['batch_size_tokens']:,}")
        print(f"  batch (seqs)        : {hpo['batch_size_seqs']:,}")
```

### C-1. ベースラインアーキテクチャの実装

```python
# models/modern_gpt.py
# IMU-1 / modded-nanogpt の構成を参考にした実装

from dataclasses import dataclass, field
from typing import Optional
import torch
import torch.nn as nn
import torch.nn.functional as F

@dataclass
class ModernGPTConfig:
    # ─── スケール ───
    vocab_size : int = 32768
    n_layer    : int = 8
    n_head     : int = 12
    n_kv_head  : int = 4         # GQA: KV ヘッド数（n_head と同じなら通常の MHA）
    n_embd     : int = 768
    block_size : int = 512
    ffn_mult   : float = 8/3     # SwiGLU の FFN hidden = n_embd * ffn_mult

    # ─── アーキテクチャ変更フラグ（ablation 用） ───
    use_qk_norm           : bool = True   # QK-Norm（最重要、-0.63%）
    use_value_residual    : bool = True   # Normalized Value Residual（-0.38%）
    use_layernorm_scaling : bool = True   # LayerNorm Scaling（-0.36%）
    use_per_head_gating   : bool = True   # Per-Head Gating（-0.05%）

    # ─── RoPE ───
    rope_theta : float = 10000.0

    # ─── 正則化 ───
    z_loss_weight : float = 1e-4   # Z-loss (logit magnitude の抑制)


class RMSNorm(nn.Module):
    def __init__(self, dim: int, eps: float = 1e-6):
        super().__init__()
        self.eps = eps
        self.weight = nn.Parameter(torch.ones(dim))

    def forward(self, x):
        norm = x.pow(2).mean(-1, keepdim=True).add(self.eps).rsqrt()
        return x * norm * self.weight


class RotaryEmbedding(nn.Module):
    def __init__(self, dim: int, theta: float = 10000.0):
        super().__init__()
        inv_freq = 1.0 / (theta ** (torch.arange(0, dim, 2).float() / dim))
        self.register_buffer('inv_freq', inv_freq)

    def forward(self, seq_len: int, device):
        t = torch.arange(seq_len, device=device).float()
        freqs = torch.outer(t, self.inv_freq)
        return torch.cat([freqs, freqs], dim=-1)  # (T, dim)


def rotate_half(x):
    x1, x2 = x.chunk(2, dim=-1)
    return torch.cat([-x2, x1], dim=-1)


def apply_rotary(x, cos, sin):
    # x: (B, H, T, d_head)
    cos = cos.unsqueeze(0).unsqueeze(0)  # (1,1,T,d_head)
    sin = sin.unsqueeze(0).unsqueeze(0)
    return x * cos + rotate_half(x) * sin


class CausalSelfAttention(nn.Module):
    def __init__(self, config: ModernGPTConfig):
        super().__init__()
        self.config = config
        self.n_head    = config.n_head
        self.n_kv_head = config.n_kv_head
        self.d_head    = config.n_embd // config.n_head

        # Q, K, V, O プロジェクション
        self.q_proj = nn.Linear(config.n_embd, config.n_embd, bias=False)
        self.k_proj = nn.Linear(config.n_embd, self.n_kv_head * self.d_head, bias=False)
        self.v_proj = nn.Linear(config.n_embd, self.n_kv_head * self.d_head, bias=False)
        self.o_proj = nn.Linear(config.n_embd, config.n_embd, bias=False)

        # QK-Norm（-0.63% loss）
        if config.use_qk_norm:
            self.q_norm = RMSNorm(self.d_head)
            self.k_norm = RMSNorm(self.d_head)
            self.qk_scale = nn.Parameter(torch.ones(1))

        # Per-Head Gating（-0.05% loss、QK-Norm との相乗効果あり）
        if config.use_per_head_gating:
            self.gate_proj = nn.Linear(config.n_embd, config.n_head, bias=True)

        # Value Residual（-0.38% loss）
        if config.use_value_residual:
            self.v_residual_alpha = nn.Parameter(torch.ones(1))
            self.v_residual_beta  = nn.Parameter(torch.zeros(1))
            self.v_residual_scale = nn.Parameter(torch.ones(1))

        self.rotary = RotaryEmbedding(self.d_head, config.rope_theta)

    def forward(self, x, v_first: Optional[torch.Tensor] = None):
        B, T, C = x.shape

        q = self.q_proj(x).view(B, T, self.n_head,    self.d_head).transpose(1, 2)
        k = self.k_proj(x).view(B, T, self.n_kv_head, self.d_head).transpose(1, 2)
        v = self.v_proj(x).view(B, T, self.n_kv_head, self.d_head).transpose(1, 2)

        # QK-Norm
        if self.config.use_qk_norm:
            q = self.q_norm(q)
            k = self.k_norm(k)

        # RoPE
        cos_sin = self.rotary(T, x.device)
        cos = cos_sin.cos()
        sin = cos_sin.sin()
        q = apply_rotary(q, cos, sin)
        k = apply_rotary(k, cos, sin)

        # Value Residual: V = scale * (α1*V_local + α2*V_first) / sqrt(α1²+α2²)
        if self.config.use_value_residual and v_first is not None:
            a1, a2 = self.v_residual_alpha, self.v_residual_beta
            norm = (a1 ** 2 + a2 ** 2).sqrt().clamp(min=1e-6)
            v_first_kv = v_first.view(B, T, self.n_kv_head, self.d_head).transpose(1, 2)
            v = self.v_residual_scale * (a1 * v + a2 * v_first_kv) / norm

        # GQA: KV を n_head に展開
        if self.n_kv_head != self.n_head:
            repeat = self.n_head // self.n_kv_head
            k = k.repeat_interleave(repeat, dim=1)
            v = v.repeat_interleave(repeat, dim=1)

        # scaled dot-product attention（Flash Attention が使えれば自動的に使われる）
        scale = self.qk_scale if self.config.use_qk_norm else None
        attn = F.scaled_dot_product_attention(
            q, k, v,
            attn_mask=None,
            dropout_p=0.0,
            is_causal=True,
            scale=scale,
        )

        attn = attn.transpose(1, 2).contiguous().view(B, T, C)

        # Per-Head Gating
        if self.config.use_per_head_gating:
            g = torch.sigmoid(self.gate_proj(x))  # (B, T, n_head)
            attn = attn.view(B, T, self.n_head, self.d_head)
            attn = 2.0 * g.unsqueeze(-1) * attn
            attn = attn.view(B, T, C)

        return self.o_proj(attn)


class SwiGLUFFN(nn.Module):
    def __init__(self, config: ModernGPTConfig):
        super().__init__()
        hidden = int(config.n_embd * config.ffn_mult)
        hidden = (hidden + 63) // 64 * 64  # 64 の倍数に切り上げ
        self.w1 = nn.Linear(config.n_embd, hidden, bias=False)
        self.w2 = nn.Linear(hidden, config.n_embd, bias=False)
        self.w3 = nn.Linear(config.n_embd, hidden, bias=False)

    def forward(self, x):
        return self.w2(F.silu(self.w1(x)) * self.w3(x))


class TransformerBlock(nn.Module):
    def __init__(self, config: ModernGPTConfig, layer_idx: int):
        super().__init__()
        self.layer_idx = layer_idx
        self.n_layer   = config.n_layer
        self.config    = config

        self.attn_norm = RMSNorm(config.n_embd)
        self.ffn_norm  = RMSNorm(config.n_embd)
        self.attn      = CausalSelfAttention(config)
        self.ffn       = SwiGLUFFN(config)

        # LayerNorm Scaling: 深いレイヤーの寄与を 1/√l で補正（-0.36% loss）
        if config.use_layernorm_scaling:
            self.ln_scale = 1.0 / ((layer_idx + 1) ** 0.5)
        else:
            self.ln_scale = 1.0

    def forward(self, x, v_first=None):
        normed = self.attn_norm(x) * self.ln_scale
        x = x + self.attn(normed, v_first=v_first)
        x = x + self.ffn(self.ffn_norm(x) * self.ln_scale)
        return x


class ModernGPT(nn.Module):
    def __init__(self, config: ModernGPTConfig):
        super().__init__()
        self.config = config
        self.tok_emb = nn.Embedding(config.vocab_size, config.n_embd)
        self.blocks  = nn.ModuleList([
            TransformerBlock(config, i) for i in range(config.n_layer)
        ])
        self.norm_f  = RMSNorm(config.n_embd)
        self.lm_head = nn.Linear(config.n_embd, config.vocab_size, bias=False)

        # weight tying（embedding と lm_head を共有）
        self.lm_head.weight = self.tok_emb.weight

        self.apply(self._init_weights)

    def _init_weights(self, module):
        if isinstance(module, nn.Linear):
            nn.init.normal_(module.weight, mean=0.0, std=0.02)
            if module.bias is not None:
                nn.init.zeros_(module.bias)
        elif isinstance(module, nn.Embedding):
            nn.init.normal_(module.weight, mean=0.0, std=0.02)

    def forward(self, idx, targets=None):
        B, T = idx.shape
        x = self.tok_emb(idx)

        # Value Residual のために第1層の V を保存
        v_first = None
        for i, block in enumerate(self.blocks):
            if i == 0 and self.config.use_value_residual:
                # 第1層の v_proj 出力を v_first として保存
                v_first = block.attn.v_proj(block.attn_norm(x))
            x = block(x, v_first=v_first)

        x = self.norm_f(x)
        logits = self.lm_head(x)

        loss = None
        if targets is not None:
            loss = F.cross_entropy(
                logits.view(-1, logits.size(-1)),
                targets.view(-1),
            )
            # Z-loss: logit magnitude を抑制
            if self.config.z_loss_weight > 0:
                z = torch.logsumexp(logits, dim=-1)
                loss = loss + self.config.z_loss_weight * z.pow(2).mean()

        return logits, loss
```

### C-2. NorMuon オプティマイザの実装

```python
# optimizers/normuon.py
# 論文: arXiv:2510.05491 (NorMuon)
# Newton-Schulz 直交化 + neuron-wise normalization

import torch

def newton_schulz_orthogonalize(G: torch.Tensor, steps: int = 7) -> torch.Tensor:
    """
    Newton-Schulz 反復で行列 G を直交行列に近似する。
    Polar Express 定数 (Amsel et al. 2025) を使用。
    
    入力 G: (m, n) ただし m >= n とする（必要なら転置）
    出力: G に近い半直交行列
    """
    assert G.ndim == 2
    m, n = G.shape
    transposed = m < n
    if transposed:
        G = G.T
        m, n = n, m

    # Polar Express 定数（7ステップ用）
    # Muon の実装では係数 a, b, c を使う 5次多項式反復
    # G <- a*G + b * G @ G.T @ G + c * G @ G.T @ G @ G.T @ G
    a, b, c = (3.4445, -4.7750, 2.0315)

    X = G / (G.norm() + 1e-7)
    for _ in range(steps):
        A = X @ X.T
        X = a * X + b * (A @ X) + c * (A @ A @ X)

    if transposed:
        X = X.T
    return X


def neuron_norm(update: torch.Tensor, eps: float = 1e-8) -> torch.Tensor:
    """
    Neuron-wise normalization:
    各出力ニューロン（行）の更新ベクトルをそのノルムで割る。
    """
    row_norms = update.norm(dim=1, keepdim=True).clamp(min=eps)
    return update / row_norms


class NorMuon(torch.optim.Optimizer):
    """
    NorMuon: Muon + neuron-wise adaptive learning rates
    
    2D パラメータ（weight matrices）に適用。
    1D パラメータ（embedding, bias, LN）は AdamW で別途最適化。
    
    Args:
        params: 2D パラメータのみを渡すこと
        lr: 学習率（デフォルト: 0.0235、IMU-1 実験値）
        momentum: モーメンタム係数（デフォルト: 0.95）
        ns_steps: Newton-Schulz 反復ステップ数（デフォルト: 7）
        weight_decay: 通常の weight decay 係数
        cautious: Cautious Weight Decay を使用するか
    """
    def __init__(self, params, lr=0.0235, momentum=0.95,
                 ns_steps=7, weight_decay=0.1, cautious=True):
        defaults = dict(lr=lr, momentum=momentum, ns_steps=ns_steps,
                        weight_decay=weight_decay, cautious=cautious)
        super().__init__(params, defaults)

    @torch.no_grad()
    def step(self):
        for group in self.param_groups:
            lr           = group['lr']
            momentum     = group['momentum']
            ns_steps     = group['ns_steps']
            weight_decay = group['weight_decay']
            cautious     = group['cautious']

            for p in group['params']:
                if p.grad is None:
                    continue

                g = p.grad
                assert g.ndim == 2, \
                    "NorMuon は 2D パラメータ専用。1D は AdamW を使用してください。"

                state = self.state[p]
                if 'momentum_buffer' not in state:
                    state['momentum_buffer'] = torch.zeros_like(g)

                buf = state['momentum_buffer']
                buf.mul_(momentum).add_(g)

                # Newton-Schulz 直交化
                update = newton_schulz_orthogonalize(buf, steps=ns_steps)

                # Neuron-wise normalization
                update = neuron_norm(update)

                # Cautious Weight Decay: update と weight の符号が一致する場合のみ decay
                if weight_decay > 0:
                    if cautious:
                        # sign(update) == sign(p) の要素にのみ decay を適用
                        mask = (update.sign() == p.sign()).float()
                        p.data.mul_(1 - lr * weight_decay * mask)
                    else:
                        p.data.mul_(1 - lr * weight_decay)

                p.data.add_(update, alpha=-lr)
```

### C-3. WSD スケジューラの実装

```python
# schedulers/wsd.py

import math

class WSDScheduler:
    """
    Warmup-Stable-Decay スケジューラ（1-√t decay profile）。
    
    IMU-1 実験より:
      - stable LR = peak LR × 0.55
      - decay fraction = 20% が最適（10%は不十分, 30%は過剰）
      - min_lr = 1e-5（固定定数。max_lr/10 ではない）
    
    Args:
        optimizer      : PyTorch オプティマイザ
        peak_lr_2d     : NorMuon (2D) のピーク LR
        peak_lr_1d     : AdamW (1D) のピーク LR  
        total_steps    : 全学習ステップ数
        warmup_frac    : Warmup 割合（デフォルト: 1%）
        decay_frac     : Decay 割合（デフォルト: 20%）
        min_lr         : 最小 LR（デフォルト: 1e-5 固定）
        stable_ratio   : stable LR = peak × stable_ratio（デフォルト: 0.55）
    """
    def __init__(self, optimizer, peak_lr_2d, peak_lr_1d,
                 total_steps, warmup_frac=0.01, decay_frac=0.20,
                 min_lr=1e-5, stable_ratio=0.55):
        self.optimizer    = optimizer
        self.peak_lrs     = [peak_lr_2d, peak_lr_1d]   # param_group の順序に対応
        self.stable_lrs   = [lr * stable_ratio for lr in self.peak_lrs]
        self.min_lr       = min_lr
        self.total_steps  = total_steps
        self.warmup_steps = int(total_steps * warmup_frac)
        self.decay_start  = int(total_steps * (1 - decay_frac))
        self.decay_steps  = total_steps - self.decay_start
        self._step        = 0

    def step(self):
        self._step += 1
        lrs = self._get_lrs()
        for pg, lr in zip(self.optimizer.param_groups, lrs):
            pg['lr'] = lr
        return lrs

    def _get_lrs(self):
        s = self._step
        result = []
        for peak_lr, stable_lr in zip(self.peak_lrs, self.stable_lrs):
            if s <= self.warmup_steps:
                # Linear warmup → peak
                t = s / max(1, self.warmup_steps)
                lr = self.min_lr + (peak_lr - self.min_lr) * t
            elif s <= self.decay_start:
                # Stable phase（peak × 0.55 で固定）
                lr = stable_lr
            else:
                # 1 - √t decay
                t = (s - self.decay_start) / self.decay_steps
                decay = 1.0 - math.sqrt(t)
                lr = self.min_lr + (stable_lr - self.min_lr) * decay
            result.append(lr)
        return result

    @property
    def current_lrs(self):
        return [pg['lr'] for pg in self.optimizer.param_groups]
```

### D-2. Step Law 予測精度の記録

```python
# evaluation/step_law_validation.py

def record_step_law_accuracy(
    n_params: int,
    n_tokens: int,
    predicted_lr: float,
    actual_best_lr: float,     # グリッドサーチで確認した実際の最良 LR（任意）
    final_loss: float,
    chinchilla_loss: float,    # 20:1 で学習した場合の loss（比較用）
) -> dict:
    """
    Step Law の予測精度を記録。Phase 1 移行判断に使う。
    """
    lr_error_pct = abs(predicted_lr - actual_best_lr) / actual_best_lr * 100
    return {
        'n_params'       : n_params,
        'n_tokens'       : n_tokens,
        'predicted_lr'   : predicted_lr,
        'actual_best_lr' : actual_best_lr,
        'lr_error_pct'   : lr_error_pct,
        'final_loss'     : final_loss,
        'chinchilla_loss': chinchilla_loss,
        'data_efficiency': chinchilla_loss / final_loss,   # >1 なら改善
    }
```
